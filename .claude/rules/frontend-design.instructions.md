This skill guides creation of distinctive, production-grade frontend interfaces that avoid generic "AI slop" aesthetics. Implement real working code with exceptional attention to aesthetic details and creative choices.

The user provides frontend requirements: a component, page, application, or interface to build. They may include context about the purpose, audience, or technical constraints.

## Design Thinking

Before coding, understand the context and commit to a BOLD aesthetic direction.

For this skill, the **default direction should lean toward Brutalist UI design** unless the user explicitly asks for a different aesthetic.

Brutalist UI characteristics to emphasize:

- Raw, unapologetic visual hierarchy
- Thick borders and hard edges
- High contrast color palettes
- Utility-focused layout
- Visible structure and grids
- Minimal decorative softness
- Loud typography and oversized elements
- Intentional asymmetry and visual tension

Design should feel **honest, bold, slightly aggressive, and highly functional** rather than polished or corporate.

Consider:

- **Purpose**: What problem does this interface solve? Who uses it?
- **Tone**: Brutalist, industrial, editorial, hacker-style, utilitarian web. Interfaces should feel bold and uncompromising.
- **Constraints**: Technical requirements (framework, performance, accessibility).
- **Differentiation**: What makes this UNFORGETTABLE? What's the one thing someone will remember?

**CRITICAL**: Choose a clear conceptual direction and execute it with precision. Brutalism should feel intentional and structured, not messy.

Then implement working code (HTML/CSS/JS, React, Vue, etc.) that is:

- Production-grade and functional
- Visually striking and memorable
- Cohesive with a clear aesthetic point-of-view
- Meticulously refined in every detail

## Frontend Aesthetics Guidelines

Focus on:

### Typography

Typography should feel **bold and unapologetic**.

- Prefer strong grotesk, monospace, or editorial fonts.
- Large headings, tight tracking, aggressive scale.
- Avoid soft corporate fonts.
- Typography should dominate the layout.

### Color & Theme

- High contrast palettes.
- Limited color usage (often 1–3 colors).
- Stark combinations like:
  - black / white
  - neon accent colors
  - red / black
  - yellow / black
- Use CSS variables for consistency.

Brutalist design prioritizes **impact over harmony**.

### Motion

Motion should be **functional and sharp**, not soft or decorative.

- Fast transitions
- Hard hover states
- Sharp reveals
- Minimal easing

Avoid overly smooth or "luxury" animation styles.

### Spatial Composition

Layouts should emphasize **structure and tension**:

- Hard grid systems
- Thick borders
- Block-based composition
- Asymmetric alignment
- Large spacing contrasts
- Elements that intentionally break visual balance

### Backgrounds & Visual Details

Avoid polished gradients and glossy effects.

Prefer:

- Flat surfaces
- Grid overlays
- Noise or print-like textures
- Harsh shadows
- Outline-based components
- Thick borders
- Sharp containers

Interfaces should feel **structural and architectural** rather than decorative.

## Anti-Patterns

NEVER use generic AI-generated aesthetics like:

- Overused font families (Inter, Roboto, Arial)
- Soft SaaS gradients
- Glassmorphism
- Neumorphism
- Rounded everything
- Friendly corporate UI kits
- Cookie-cutter dashboards

Avoid predictable layouts and component patterns that lack personality.

## Implementation Philosophy

Interpret creatively and make unexpected choices that feel genuinely designed for the context.

Brutalist interfaces should feel:

- Raw but deliberate
- Functional but expressive
- Simple but memorable
- Slightly confrontational in visual hierarchy

**IMPORTANT**: Match implementation complexity to the aesthetic vision.

Brutalist design typically relies on:

- Strong layout structure
- Typography-driven hierarchy
- Thick borders and containers
- High contrast styling
- Minimal but purposeful motion

Elegance comes from executing the bold vision with precision.

Remember: Claude is capable of extraordinary creative work. Don't hold back, show what can truly be created when thinking outside the box and committing fully to a distinctive brutalist interface.
